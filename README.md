# Retail-Sales-Intelligence-Dashboard



##  Project Overview: 
This project analyzes sales data from the **Sample Superstore dataset** using **Python, Pandas, Plotly, and Streamlit**.  

It provides interactive dashboards for:

- Category-wise and Region-wise sales
  
- Time series analysis of monthly sales

- Segment and sub-category breakdowns

- Profit vs Sales scatter plots

- Hierarchical treemap visualization



##  Requirements :
Install dependencies using `requirements.txt`:

pip install -r requirements.txt
